# Fairyfingershair Braiding Website

Production-ready static website for Fairyfingershair Braiding.

## Stack
- HTML5 + CSS3 + vanilla JS
- SEO metadata + JSON-LD schema
- Responsive and accessible baseline
- GitHub Pages ready

## Local preview
Open `index.html` directly, or run any local static server.

## Deploy (GitHub Pages)
1. Push to GitHub repository.
2. Enable Pages from `main` branch root (`/`).
3. Site URL format:
   - `https://<username>.github.io/<repo>/`
